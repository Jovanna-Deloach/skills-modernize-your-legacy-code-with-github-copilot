## Notes for Node.js Modernization

- Use each test case as an acceptance test scenario to preserve current COBOL behavior.
- Convert arithmetic-oriented cases (credits, debits, boundary values) into unit tests.
- Convert menu-to-operation and read/write flow cases into integration tests.
- Keep `Actual Result`, `Status (Pass/Fail)`, and `Comments` as execution-time fields for stakeholder sign-off.
